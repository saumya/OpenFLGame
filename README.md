An OpenFL Component library
==========================
The following are the details of the components in this library.


 - ButtonWithRandomColor
 - ButtonWithBgColor
 - Button with Fixed width

```
// new game    : 
openfl create project MyGame
//
openfl build neko
openfl run neko
//
openfl test neko
```
